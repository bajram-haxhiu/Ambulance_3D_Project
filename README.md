# Ambulance 3D Project

## Overview

Ambulance 3D Project is a Java-based application developed as a university project.  
The main purpose of this project is to simulate and visualize an ambulance system in a 3D environment.

The project focuses on representing ambulance movement, route visualization, and emergency response logic using Java.  
It was developed using the Eclipse IDE.

---

## Project Description

This project demonstrates how an ambulance system can be modeled using programming concepts and object-oriented design.

The application includes source code for managing the ambulance simulation and provides a structured Java project that can be opened and executed in Eclipse.

The project may be used for academic purposes, demonstrations, and learning about Java-based simulations.

---

## Main Features

- Java-based ambulance simulation
- 3D-style project structure and visualization concept
- Organized source code inside the `src` folder
- Eclipse project configuration included
- Easy to import and run in Eclipse
- Suitable for university project presentation

---

## Technologies Used

- Java
- Eclipse IDE
- Object-Oriented Programming concepts
