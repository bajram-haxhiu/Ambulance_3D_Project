# Ambulance 3D Project

## Overview

Ambulance 3D Project is a Java-based project developed for the **Data Structures and Algorithms (DSA)** course.

The main purpose of this project is to simulate an ambulance route/pathfinding system and demonstrate how data structures and algorithms can be used to solve real-world problems.

The project focuses on ambulance movement, route visualization, graph-based navigation, and emergency response logic using Java.

---

## Project Description

This project represents an ambulance system where routes and locations can be modeled using graph concepts.

The system is designed to show how algorithms can help find and analyze routes in an emergency transportation scenario.
It demonstrates the use of programming, object-oriented design, and algorithmic thinking in a practical Java application.

The project was developed using the Eclipse IDE and is organized as a Java project.

## How to Run the Project

To run this project, follow these steps:

1. Download or clone the repository from GitHub.
2. Open Eclipse IDE.
3. Click **File**.
4. Select **Import**.
5. Choose **Existing Projects into Workspace**.
6. Click **Next**.
7. Browse and select the project folder.
8. Click **Finish**.
9. Open the `src` folder.
10. Find the main Java class.
11. Right-click the main class.
12. Select **Run As → Java Application**.
